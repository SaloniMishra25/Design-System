This project is a reusable, scalable "Design System" built with:

- 🧩 React (with TypeScript)
- 🎨 TailwindCSS (custom theme & tokens)
- 📖 Storybook (for documentation)
- ♿ Accessible components
- 📱 Responsive layouts

🚀 Getting Started

bash
# Install dependencies
npm install

# Run the app
npm start

# Run Storybook
npm run storybook
```
